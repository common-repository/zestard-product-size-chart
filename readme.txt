== Description ==

Zestard Product Size Chart helps you to create custom size charts for any products and categories of woocommerce.

You can customize your product size chart table as you prefer, and you might use it even for clothes, furniture measures or any other products that comes to your mind.

== Features ==

- Allows you to create multiple custom Product size charts.
- Allows you to assign created Product Size Chart to a multiple product as well as multiple categories.
- Your created Product Size Chart will display in a custom tab on single product page.

== Installation ==

1. Upload the `zestard-product-size-chart` folder to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Go to "Woocommerce" menu in wordpress

4. Click on 'Zestard Size Charts'.

5. Add new Size Chart title and create size chart table.

6. Select product or category for assign to display on single product page.

7. Click on "Save Changes" button.

== Display Your Size Chart ==

You can view the size chart table in product single page inside the product tab.
